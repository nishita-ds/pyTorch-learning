# PyTorch Neural Network (nn Module)

This project demonstrates the implementation of neural networks using the PyTorch nn module.
It covers the fundamentals of building, training, and evaluating deep learning models in a 
structured and efficient way.

## Overview

The torch.nn module provides a high-level API to build neural networks.
This project focuses on:

>Creating custom neural network architectures
>Using built-in layers like Linear, ReLU, Sigmoid
>Defining loss functions
>Training using backpropagation and optimizers
